## ✨ Features

- 🎮 **3D Interactive Background** (Three.js)
- 🎤 **Voice Input** (Speech Recognition)
- 🤖 **Groq AI** (Free, Fast, Llama 3.3 70B)
- 🔐 **Firebase Authentication** (Email/Password + Google Login)
- 📱 **ApkClaw Integration** (Phone lock suggestions)
- 🎬 **Cinematic Intro Animation**
- 📱 **Fully Responsive** (Phone, Tablet, Laptop)

## 🚀 Live Demo

(https://snockyaidost.netlify.app/)
## 🛠️ Tech Stack

- Frontend: HTML5, CSS3, JavaScript
- 3D Graphics: Three.js
- AI Backend: Groq Cloud (Llama 3.3 70B)
- Authentication: Firebase Auth
- Database: Firestore
- Hosting: GitHub Pages

## 📦 Setup

1. Clone the repo
2. Add your Firebase config in `index.html`
3. Deploy to GitHub Pages
